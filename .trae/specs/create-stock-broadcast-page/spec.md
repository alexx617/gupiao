# 股票播报网页 Spec

## Why
用户需要一个能够自动播报所选股票价格的网页应用，方便实时监控股票行情，无需手动刷新页面。

## What Changes
- 创建基于 Vue3 的股票播报单页应用
- 实现股票列表管理功能（添加、删除股票）
- 实现每分钟自动播报股票价格功能
- 展示股票详细信息面板

## Impact
- 新建项目：Vue3 单页应用
- 使用 Element Plus 作为 UI 组件库
- 使用 Web Speech API 实现语音播报

## ADDED Requirements

### Requirement: 股票列表管理
系统应提供股票列表管理功能，用户只需输入股票代码，系统自动获取并显示股票名称，支持添加和删除操作。

#### Scenario: 添加股票
- **WHEN** 用户在输入框中输入股票代码并点击添加按钮
- **THEN** 系统调用API获取该股票信息，自动显示股票名称，并将该股票添加到监控列表

#### Scenario: 自动获取股票名称
- **WHEN** 用户输入有效的股票代码后
- **THEN** 系统自动调用API获取股票名称并显示在列表中

#### Scenario: 删除股票
- **WHEN** 用户点击股票列表中某只股票的删除按钮
- **THEN** 系统从监控列表中移除该股票，并更新表格显示

#### Scenario: 无效股票代码处理
- **WHEN** 用户输入无效的股票代码
- **THEN** 系统显示错误提示，不添加到列表中

### Requirement: 股票列表展示
系统应以表格形式展示用户添加的所有股票列表，包含股票代码、股票名称和操作按钮。

#### Scenario: 显示股票列表
- **WHEN** 用户添加股票后
- **THEN** 系统在表格中展示股票代码、股票名称、删除按钮

### Requirement: 股票详情数据展示
系统应以表格形式展示每只股票的详细行情数据，数据实时更新。

#### Scenario: 显示行情数据表格
- **WHEN** 股票数据获取成功
- **THEN** 系统以表格形式展示以下信息：
  - 股票代码
  - 股票名称
  - 最新价格（元）
  - 行情时间
  - 涨跌幅（%）
  - 涨速（%）
  - 成交量（手）
  - 成交额（元）
  - 开盘价（元）
  - 最高价（元）
  - 最低价（元）
  - 昨收价（元）
  - 振幅（%）
  - 换手率（%）
  - 量比
  - 委比（%）
  - 内盘（手）
  - 外盘（手）
  - 均价（元）

### Requirement: 自动播报功能
系统应提供自动语音播报功能，支持用户自定义播报间隔时间，依次播报所选股票的最新价格。

#### Scenario: 设置播报间隔
- **WHEN** 用户在播报间隔输入框中输入分钟数
- **THEN** 系统保存该设置，并按设置的间隔时间进行播报

#### Scenario: 开启播报
- **WHEN** 用户点击开启播报按钮
- **THEN** 系统开始按设置的间隔时间依次播报列表中所有股票的名称和最新价格

#### Scenario: 关闭播报
- **WHEN** 用户点击关闭播报按钮
- **THEN** 系统停止自动播报

#### Scenario: 默认播报间隔
- **WHEN** 用户未设置播报间隔
- **THEN** 系统默认每1分钟播报一次

### Requirement: 数据刷新
系统应定期刷新股票数据以保持信息最新。

#### Scenario: 定时刷新
- **WHEN** 系统运行中
- **THEN** 每分钟自动刷新所有监控股票的数据

## API 接口说明

### 接口地址
```
GET http://www.sanhulianghua.com:2008/v1/hsa_fenshi
```

### 请求参数
| 参数名称 | 描述 | 类型 | 默认值 | 是否必须 | 取值范围 | 示例值 |
|----------|------|------|--------|----------|----------|--------|
| token | 会员令牌 | string | - | 是 | 非空，来自【会员信息】 | f75cc676f91464a63112979b855c4164 |
| code | 股票代码 | string | - | 是 | 非空 | 601857 |
| all | 是否获取全日数据 | uint | 0 | 否 | 0：获取最新数据；1：获取全日数据 | 1 |

### 返回码说明
| 返回码 | 描述 |
|--------|------|
| 200 | 成功 |
| 201 | 盘前无数据 |
| 300 | 令牌错误 |
| 301 | 余额不足 |
| 302 | 股票代码无效 |
| 4XX | 其他错误 |

### 数据转换规则
- **价格转换**：实际价格 = 接口值 / 10（单位：分）或 / 100（单位：元）
- **百分比转换**：实际百分比 = 接口值 / 1000（如涨幅、振幅、换手等）
- **市值转换**：实际市值 = 接口值（单位：万）
- **成交量转换**：实际成交量 = 接口值（单位：手）
- **成交额转换**：实际成交额 = 接口值（单位：元）

## 技术方案

### 前端框架
- Vue 3 + Composition API
- Vuex 状态管理
- Vue Router 路由
- Element Plus UI 组件库
- Axios HTTP 请求
- SCSS 样式

### 语音播报
- 使用 Web Speech API 的 SpeechSynthesis 接口
- 支持中文语音播报

### 数据存储
- 使用 localStorage 存储用户添加的股票列表
- 页面刷新后自动恢复股票列表
